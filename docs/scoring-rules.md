# T20 Fantasy Points Scoring Rules

## Batting

| Category              | Points              |
| --------------------- | ------------------- |
| Run scored            | +1 per run          |
| Boundary (4)          | +1 bonus per four   |
| Six                   | +2 bonus per six    |
| Half-century (50)     | +8 bonus            |
| Century (100)         | +16 bonus           |
| 150+                  | +24 bonus           |
| Duck (0 runs, out)    | -2 (non-bowlers)    |

### Strike Rate Bonus/Penalty (minimum 10 balls faced)

| Strike Rate     | Points |
| --------------- | ------ |
| > 170           | +6     |
| 150.01 – 170    | +4     |
| 130.01 – 150    | +2     |
| 60.01 – 70      | -2     |
| 50.01 – 60      | -4     |
| ≤ 50            | -6     |

## Bowling

| Category              | Points              |
| --------------------- | ------------------- |
| Wicket                | +25 per wicket      |
| LBW / Bowled bonus    | +8 per dismissal    |
| 3-wicket haul         | +4 bonus            |
| 4-wicket haul         | +8 bonus            |
| 5-wicket haul         | +16 bonus           |
| Maiden over           | +12 per maiden      |
| Dot ball              | +1 per dot          |

### Economy Rate Bonus/Penalty (minimum 2 overs bowled)

| Economy Rate    | Points |
| --------------- | ------ |
| < 5             | +6     |
| 5 – 5.99       | +4     |
| 6 – 7          | +2     |
| 10.01 – 11     | -2     |
| 11.01 – 12     | -4     |
| > 12            | -6     |

## Fielding

| Category              | Points              |
| --------------------- | ------------------- |
| Catch                 | +8 per catch        |
| Stumping              | +12                 |
| Run-out (direct)      | +12                 |
| Run-out (indirect)    | +6                  |
| 3+ catches bonus      | +4                  |
